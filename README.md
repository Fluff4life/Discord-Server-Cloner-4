## Description:
This is the software to clone any discord server as your own

## Features:
<ul>
  <li>Clone server's logo</li>
  <li>Clone server's emojis</li>
  <li>Clone roles</li>
  <li>Clone categories</li>
  <li>Clone channels</li>
  <li>Clone all permissions</li>
</ul>
